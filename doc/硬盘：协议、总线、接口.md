# 协议
目前常用的硬盘协议有：IDE、AHCI、NVMe、SCSI。其中，前三者主要是民用场合，最后一种是服务器领域。
# 总线
目前硬盘常用的总线有：SATA、PCIe、SAS
# 接口
目前，常用的硬盘接口有：SATA、mSATA、SATA Express（特快）、M.2（具体分为：B-key & M-key）、PCIe、U.2、SAS接口。
